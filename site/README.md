# VRCX-Luo intro site

这是一个可以直接部署到 Cloudflare Pages 的静态介绍页。

## Cloudflare Pages 设置

- Framework preset: `None`
- Build command: 留空
- Build output directory: `site`

页面入口是 `site/index.html`，仓库按钮指向：

- `https://github.com/yixijun/VRCX-Luo`
- `https://github.com/yixijun/VRCX-Luo/releases/latest`

页面已注明：

- VRCX-Luo 是 `vrcx-team/VRCX` 的社区 fork
- 本分支包含 AI 辅助修改
- 本项目不隶属于 VRChat Inc.，也不是 VRCX 官方版本
